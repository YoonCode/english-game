패턴 정리
패턴 1
Can you + 동사?도움이나 정보를 부탁할 때. 쇼핑 상황에서 "Can you show me ~?" / "Can you check ~?" 자주 써요.
패턴 2
I'd like to + 동사원하는 것을 정중하게 말할 때. "I want to"보다 더 부드럽고 자연스러운 표현이에요.
패턴 3
I think / I don't think 주어 + 동사의견을 말할 때. "I think it's ~" / "I don't think I need ~"처럼 확신보다 부드럽게 표현해요.
